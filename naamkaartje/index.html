Fietspas

Markeer leerlingen die een schoolfiets mogen lenen. Zelfde badge, andere kleur — één klik.

Naamkaartjes

Grote, leesbare naamkaartjes voor evenementen en ouderavonden. 10 per A4, 90 × 55 mm.

🎨

Volledig aanpasbaar

Schoolnaam, logo, kleuren, eigen lettertype (TTF) en het formaat van het stickervel zijn vrij instelbaar.

🔒

Privacy by design

Geen account, geen cloud. Alle data — foto's, namen, CSV's — blijft lokaal op jouw toestel.

📄

Slim veldbeheer

De app houdt bij welke stickers al gebruikt zijn. Zo verspil je nooit een gedeeltelijk vel.

Hoe gebruik je het?

1

Open de app

Klik op de knop hierboven. De app werkt rechtstreeks in je browser — geen installatie nodig.

2

Stel je school in

Ga naar Instellingen en vul je schoolnaam, website, schooljaar en logo in. Kies je accentkleur en het formaat van je stickervel.

3

Importeer leerlingen

Exporteer een CSV uit Smartschool en importeer die via Leerlingen → CSV importeren. UTF-8 en Windows-1252 worden automatisch herkend.

4

Koppel foto's

Gebruik Foto's koppelen om een map met portretfoto's te importeren. De app koppelt ze automatisch op basis van de bestandsnaam.

5

Druk af

Selecteer leerlingen in het tabblad Afdrukken, kies nieuw of bestaand vel, en klik op PDF genereren. Het bestand wordt meteen gedownload.

✦

Lokaal draaien (voor IT-beheerders)

Installeer Node.js, clone de repo en start de app:

git clone https://github.com/steinerscholen/naamkaart.git
cd naamkaart && npm install && npm run dev
