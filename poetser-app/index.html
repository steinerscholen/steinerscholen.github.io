Week­end­poets­beurten
eerlijk verdeeld

Aan het begin van het schooljaar ontvangt elke ouder een overzicht van de weekends waarop ze de klas van hun kind poetsen. Poetser genereert dat rooster automatisch — eerlijk, transparant en zonder handwerk.

Open Poetser →

Leerlingengegevens worden nooit verstuurd naar een server

📋

Automatisch rooster

Importeer de leerlingenlijst en Poetser verdeelt de beschikbare weekends gelijkmatig over alle gezinnen.

⚖️

Twee verdeelmethoden

Kies tussen een strikte roulatie of een gewogen verdeling op basis van het aantal kinderen per gezin.

👨‍👩‍👧‍👦

Gezinnen met meerdere kinderen

Ouders met kinderen in verschillende klassen poetsen niet vaker dan anderen — de app houdt hier automatisch rekening mee.

📬

Overzicht per ouder

Elke ouder krijgt een persoonlijk overzicht van hun beurt(en) voor het volledige schooljaar.

📅

Vrije weekends

Schoolvakanties en feestdagen worden automatisch uitgesloten. Je kan ook manueel weekends blokkeren.

🔒

Privacy by design

Geen account, geen cloud. Alle data — namen, klassen, roosters — blijft lokaal op jouw toestel.

Hoe gebruik je het?

1

Open de app

Klik op de knop hierboven. De app werkt rechtstreeks in je browser — geen installatie nodig.

2

Stel het schooljaar in

Geef de start- en einddatum van het schooljaar op. Schoolvakanties worden automatisch herkend en uitgesloten van het rooster.

3

Importeer leerlingen

Laad een CSV-bestand in met de leerlingenlijst van je klas. De app groepeert leerlingen automatisch per gezin op basis van het adres of de familienaam.

4

Kies de verdeelmethode

Selecteer Strikte roulatie voor een vaste beurt per gezin, of Gewogen verdeling om gezinnen met meerdere kinderen licht te begunstigen.

5

Genereer het rooster

Klik op Rooster genereren. Je ziet meteen een overzicht per weekend én per gezin. Pas eventueel manueel aan en sla op.

6

Exporteer en verstuur

Download een PDF of CSV per klas. Elk gezin krijgt een persoonlijk overzichtje dat je per e-mail of via Smartschool kan doorsturen.

✦

Lokaal draaien (voor IT-beheerders)

Installeer Node.js, clone de repo en start de app:

git clone https://github.com/steinerscholen/poetser.git
cd poetser && npm install && npm run dev
